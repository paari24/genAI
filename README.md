### ⚡ **30-Second Setup** ⚡

<div align="center">

```bash
🔥 Ultra-Fast Installation 🔥

# 1️⃣ Check Python (3.8+ Required)
python --version  # Should show 3.8+

# 2️⃣ Clone & Enter
git clone https://github.com/paari24/genAI.git
cd genAI

# 3️⃣ Magic Install ✨
pip install -r requirements.txt

# 4️⃣ Launch! 🚀
python langChain_Projects/app.py
```

</div>

### 🔧 **Advanced Setup** 🔧

<div align="center">

**🛠️ Professional Installation Pipeline 🛠️**

</div>

```bash
# 🔒 Create Isolated Environment
python -m venv paarivenv

# 🚀 Activate Environment
# Windows:
paarivenv\Scripts\activate
# Linux/Mac:
source paarivenv/bin/activate

# 📦 Install All Dependencies
pip install --upgrade pip
pip install -r requirements.txt

# 🔑 Setup Environment Variables
cp .env.example .env
# Edit .env with your API keys

# ✅ Verify Installation
python -c "import langchain; print('✅ LangChain Ready!')"
```
