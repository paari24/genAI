## 🚀 Quick Start

### Prerequisites
- Python 3.8+
- Git
- Virtual Environment

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/paari24/genAI.git
   cd genAI
   ```

2. **Set up virtual environment**
   ```bash
   python -m venv paarivenv
   # Windows
   paarivenv\Scripts\activate
   # macOS/Linux
   source paarivenv/bin/activate
   ```

3. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   # Or install core packages
   pip install langchain openai streamlit flask playwright pyautogui
   ```

4. **Set up environment variables**
   ```bash
   # Create .env file with your API keys
   echo "OPENAI_API_KEY=your_openai_key_here" > .env
   ```

5. **Install Playwright browsers**
   ```bash
   playwright install
   ```
